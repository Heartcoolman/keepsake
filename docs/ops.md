# 运维台(/ops)

部署者视角的操作员控制台:管账户、看用量、盘磁盘、控注册。页面由服务端直接托管在 `GET /ops`(与用户端 SPA 完全分离,不进用户 bundle),API 挂在 `/api/ops/*`。

## 权限模型(先读这个)

- **运维台没有任何用户密钥**。照片、日记、聊天、人脸特征、人格画像全部是密文,操作员既看不到也解不开;台面上的一切都是结构元数据(用户名、类型、条目数、磁盘占用、token 计数)。
- **不能重置用户密码**。找回只有用户自己的恢复码(`POST /api/v1/auth/recover`)——这是产品卖点,不是缺陷。
- 操作员能做的破坏性动作:停用/启用账户、**删除独立账户(含全部数据)**、**解散家庭(共享数据永久销毁)**。全部写入审计。

## 操作员账户

- 存储在 `data/ops/<id>.json`,与用户账户体系完全隔离(无密钥字段、独立的 JWT 签名域:HKDF(JWT_SECRET, "nianxiang-ops"),用户令牌与运维令牌互不接受)。
- 令牌 12 小时有效,无刷新,存于浏览器 sessionStorage;登出/改密/删号即时吊销(tokenVersion)。
- **首次引导**:在 `.env` 设 `OPS_BOOTSTRAP_TOKEN=<长随机串>` → 重启 → 打开 `/ops` → 「首次引导」输入令牌+用户名+密码。仅在操作员数为 0 时可用;引导完成后建议从 `.env` 移除令牌。
- 操作员互为对等:可互相建号、重置密码、删除(全部入审计)。
- **唯一操作员丢密码**:部署者 ssh 进服务器删除 `data/ops/` 下的记录文件,引导即重新开放。

## 功能页签

| 页签 | 内容 |
|---|---|
| 概览 | 记录计数、密钥环在线数、磁盘占用分解(60s 缓存,可强制重算)、各账户加密迁移状态、depth/analyze 缓存清理、备份管理 |
| 账户 | 全部账户结构信息(类型/家庭/条目数/占用/迁移状态);停用/启用(即时踢下线并清内存密钥);删除独立账户(硬清除,需输入用户名确认) |
| 家庭 | 家庭与成员;紧急解散(见下) |
| 用量 | AI token 消耗按月汇总与按账户明细(计费预留,当前不限额) |
| 注册 | 运行时开关注册、设置/轮换/取消注册码(写入 `data/ops-config.json`,即刻覆盖 `REGISTRATION_CODE` env;注册码不回显) |
| 操作员 | 操作员增删与密码重置 |
| 审计 | `data/ops-audit.jsonl` 尾部(操作员、动作、对象;绝不含密码/注册码明文) |

## 危险操作语义

- **删除账户**:仅限不在家庭中的账户(否则 409)。清除范围:条目+图片+缩略图、记忆/人格、月度回顾、用量计数、analyze/depth 缓存、个人作用域人物/关系/人脸缓存、指向其的邀请、账户记录本体。家庭成员须先由家庭主在应用内移出(那条路径会做 FK 轮换),或整体解散家庭。
- **解散家庭**:无密钥的紧急操作。共享作用域的密文(人物库**含全体成员的人脸建档**、关系图、人脸缓存)随密钥一同作废、直接删除,**无法恢复**;成员回到独立状态,条目重置待重扫,下次登录自动重建空白身份人物并重扫人脸(NAS 上多人同时登录会排队,属预期)。家庭主保留家庭账户类型,可在应用内「创建家庭」重新开始。幂等:中途崩溃后对同一 id 重新调用即可完成收尾。
- **备份**:复制 `data/` + `cache/`(含人脸缓存——它们不可再生,正是要备的)到 `data/backups/<时间戳>/`,在挂载卷内,容器重建不丢;内容本就是密文,备份不降低隐私。对并发写容错(跳过瞬时文件)。**没有恢复接口**:恢复=停服后手工把备份目录覆盖回 `data/`、`data/cache` 并重启。注意备份包含全部照片密文,占用与 data 同量级,记得定期清理。
- **缓存清理**:仅 depth/analyze(纯性能缓存,用户带钥会话内按需重算)。face/faceThumb 不可清——清了会破坏聚类且重扫需要属主登录。

## 部署核对清单

1. `.env` 增加 `OPS_BOOTSTRAP_TOKEN`(`openssl rand -hex 24`),重启服务。
2. 打开 `http://<host>:8787/ops` → 首次引导建操作员。
3. 从 `.env` 移除 `OPS_BOOTSTRAP_TOKEN`(可选但推荐)。
4. 若经反代暴露公网,建议对 `/ops` 与 `/api/ops` 加 IP 白名单——运维台被攻破等于全部账户可被停用/删除(仍读不到内容)。
